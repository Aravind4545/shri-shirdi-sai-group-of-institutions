# Platform User Guides

## 1. Super Admin Guide
- **Login URL**: `/admin/login`
- **Permissions**: Full control over Lakshya, Deekshya, and DAFNE programs.
- **Workflow**:
  1. Approve pending admissions in the **Admissions Management** tab.
  2. Configure tuition pricing in the **Fee Structures** tab.
  3. Monitor institution revenue and analytics on the **Reports** page.
  4. Train the AI model by uploading PDFs via the **AI Controls** interface.

## 2. Teacher Guide
- **Login URL**: `/teacher/login`
- **Permissions**: Isolated to their assigned program (e.g., A Deekshya teacher cannot see Lakshya students).
- **Workflow**:
  1. View your specific students under **My Students**.
  2. Identify weak topics affecting your batch under **Result Analytics**.
  3. Upload Mock Tests and Study Materials (Coming Soon integrations).

## 3. Student Guide
- **Login URL**: `/login`
- **Permissions**: Isolated to their own profile, learning paths, and exams.
- **Workflow**:
  1. Pay tuition installments via the **Fee Portal**.
  2. Generate a custom revision schedule using the **AI Learning Hub**.
  3. Resolve doubts 24/7 using the **Floating AI Assistant** widget.
